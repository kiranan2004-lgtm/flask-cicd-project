# Flask CI/CD Project with GitHub Actions and Render

## Project Overview

This project demonstrates a simple CI/CD pipeline using:

- Python Flask
- GitHub
- GitHub Actions
- Render Cloud Deployment

The application is automatically tested and deployed whenever code is pushed to the main branch.

---

# Features

- Flask backend application
- Automated CI pipeline using GitHub Actions
- Automatic deployment using Render
- Dynamic server-side response
- Cloud-hosted web application

# Technologies Used
